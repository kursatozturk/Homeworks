Deadline

Description

In this HW, you will implement some of the tools we have covered in the lectures. The goals of this assignment are as follows:

  • understand and implement the operations/layers in a Convolutional Neural Network.
  • apply CNN to a classification problem.
  • implement some of the mechanisms that are widely used for understanding/interpreting a trained CNN. More specifically, you will use a pre-trained CNN to show some saliency maps for a given class, generate best images for a given class (a.k.a., deep dream), and learn how to fool a CNN.

Setup

Get the code as a zip file here. As for the dependencies (similar to HW1):

Use Anaconda: The preferred approach for installing all the HW dependencies is to use Anaconda, which is a Python distribution that includes many of the most popular Python packages for science, math, engineering and data analysis. Once you install it and you're ready to go directly to working on the HW.

Download data: Once you have the starter code, you will need to download the CIFAR-10 and MNIST datasets. Run the following from the HW2 directory:

cd cs231n/datasets
./get_datasets.sh
For the CNN exercises, you will need to download another dataset and a pre-trained model. The instructions for these are outlined in the respective exercises.

Start IPython: After you have the MNIST data, you should start the IPython notebook server from the HW2 directory. If you are unfamiliar with IPython, you should read the IPython tutorial.

Submitting your work:

Once you are done working run the collectSubmission.sh script; this will produce a file called HW2.zip. Upload this file on the METUclass page of the course.

Task 1: Modular layers in CNN (20 points)

The IPython Notebook task1_layers.ipynb will walk you through implementing the layers in a CNN in a modular manner.

Task 2: Combine modular layers to train a CNN (20 points)

The IPython Notebook task2_convnet.ipynb will walk you through combining the modular layers to implement a CNN architecture.

Task 3: Image Gradients (5 points)

The IPython Notebook task3_image_gradients.ipynb will walk you through how you can use a trained CNN to compute saliency maps, and how you can fool a trained CNN.

Task 4: Image Generation (15 points)

The IPython Notebook task4_image_generation.ipynb will walk you through implementing some of the mechanisms used for understanding/interpreting a trained CNN.

Task 5: Blood Pressure from a PPG signal using CNNs (40 points)

Solve Task 4 in HW1 using CNNs this time. The problem definition is exactly the same. The only difference is that you will use a CNN this time. Build your solution in the IPython Notebook task5_blood_pressure.ipynb.
